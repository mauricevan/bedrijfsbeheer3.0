## üÜò **TROUBLESHOOTING**

### **Toast notifications not appearing?**
- Check bottom-right corner of screen
- They auto-dismiss after 3 seconds
- Try the action again

- Check if another app is using the shortcut

- Check bottom-right corner

- Data is currently mock/demo data
- Will show real data when connected to backend
- Check back after backend integration
